PARA EJECUTAR LA APLICACIÓN WEB:

1) Asegurarse de que python está instalado en la máquina
2) Clonar el repositorio
3) Instalar las dependecias que se encuentran en el requeriments.txt (pip install requeriments.txt)
4) Dentro de la carpeta "app" ejecutar el comando "python3 views.py"

PD: si da error al ejecutar el ultimo comando, asegurarse que el path de NLTK del archivo utils.py apunta al lugar correcto
